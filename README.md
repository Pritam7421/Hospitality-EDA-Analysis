🏨 Hospitality Data Analysis (EDA Project)
🚀 Overview

This project presents a comprehensive Exploratory Data Analysis (EDA) on a hospitality dataset, aimed at extracting meaningful business insights related to revenue optimization, occupancy trends, and booking behavior.

The analysis reflects a real-world scenario where data is leveraged to support strategic decision-making in the hotel industry.

🧠 Problem Statement

AtliQ Grands, a multi-city hotel chain in India, is experiencing challenges in maintaining competitive revenue and optimizing occupancy levels.

A key issue is the underutilization of available data, which limits their ability to make informed decisions.

This project focuses on analyzing booking and operational data to uncover patterns and recommend improvements.

🎯 Goals
Evaluate revenue performance across cities and hotel types
Analyze occupancy behavior and demand patterns
Assess contribution of various booking platforms
Identify and resolve data quality issues
Generate actionable insights to improve business outcomes
🔄 Workflow

Raw Data → Cleaning → Transformation → Analysis → Insights → Recommendations

🧹 Data Preparation
Eliminated invalid entries (e.g., zero or negative guest counts)
Handled missing values to maintain consistency
Identified and removed outliers using statistical methods (Standard Deviation)

✔ Ensured reliability and accuracy of the dataset for analysis

⚙️ Feature Engineering
Created a key metric: Occupancy Rate (%)
Combined multiple datasets (fact and dimension tables)
Structured data for efficient analysis

✔ Enabled comparison across locations and hotel categories

📊 Analysis Highlights
Revenue distribution by hotel category
Customer rating trends across cities
Booking platform contribution to total revenue
Weekday vs weekend occupancy comparison
Monthly performance analysis
❓ Key Business Questions
Which hotel category generates maximum revenue?
Which cities perform best in terms of occupancy?
How does demand vary between weekdays and weekends?
Which platforms drive the highest bookings?
What trends are visible in customer satisfaction?
📈 Insights
Higher weekend occupancy indicates strong leisure demand patterns
Luxury segment dominates revenue, highlighting premium market strength
Certain cities consistently outperform others in both occupancy and revenue
Heavy reliance on third-party booking platforms for revenue generation
